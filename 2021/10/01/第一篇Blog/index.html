Markdown 学习记录

网站搭建基本已经告一段落了。

到了自己书写第一篇博文的时候了，而书写的方式肯定离不开 规定的文件及其相关的操作要求。

所以首先学习Markdown文件的编写教程。

以下便是我学习的相关笔记和心得。

标题

使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。

例如”####这是一个标题“ 这个标题我们就称之为四级标题。

段落

Markdown 段落没有特殊的格式，直接编写文字就好，段落的换行是使用两个以上空格加上回车。

列表

Markdown 支持有序列表和无序列表。

无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容：

  • 第一项

有序列表使用数字并加上 . 号来表示，如：

  1. 第一项
  2. 第二项
  3. 第三项

列表嵌套

列表嵌套只需要在子列表中的选项前面添加四个空格即可：

  1. 第一项：

    • 第一项嵌套的第一个元素

    • 第一项嵌套的第二个元素

  2. 第二项：

    • 第二项嵌套的第一个元素
    • 第二项嵌套的第二个元素
